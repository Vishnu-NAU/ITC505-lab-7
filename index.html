Magical Mad Lib Adventure

Addendum

This Mad Lib lab was created using HTML for the form and an Express.js backend to handle POST requests. The form collects five inputs: a fantasy creature, magical object, verb, adjective, and place. These are used to generate a fun story, returned to the user via the server.

The Express server handles both static content and dynamic POST requests. Styling was added using CSS with a gradient background to make the page visually appealing. The form submits data back to the same page using the POST method.

This setup allows for simple and interactive server-side handling of form data using Express.
